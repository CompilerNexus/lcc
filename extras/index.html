Extras

bookidx.txt
A plain ASCII version of the index in A Retargetable C Compiler.

BUILD-LCC.sh
Shell script to automate completely the building, testing, and installation of lcc versions 3.6 and later, with no changes required for most UNIX systems. Nelson Beebe / beebe@math.utah.edu

copt.shar
Source code and documentation for copt, the simple peephole optimizer mentioned on p. 534 of A Retargetable C Compiler. This file is a shell archive; unload it with sh <copt.shar. Chris Fraser / cwfraser.webhop.net

mrst.shar
Alternative implementation of switch statements for lcc using multiway radix search trees. Ulfar Erlingsson / ulfar@acm.org
